# 📖 Overview

Car Make & Model Validator is an intelligent web application that validates vehicle details using AI and a local fallback database. It verifies vehicle make, model, and manufacturing year while providing detailed specifications, maintenance recommendations, common issues, and recall information.

The application also supports typo correction, offline simulation, telemetry logging, and validation history for a seamless user experience.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/car-make-model-validator.git

cd car-make-model-validator
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=your_api_key
```

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```

---

# ⚙️ How It Works

1. Enter the vehicle year, make, and model.
2. The application validates user input.
3. AI verifies the vehicle information.
4. If AI is unavailable, the local registry is used as a fallback.
5. Vehicle specifications and maintenance details are displayed.
6. Results are stored in local validation history.
